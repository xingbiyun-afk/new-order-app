# 移动端 UI 基线

## 色彩规范

### 主色
- **主色**：`#22BDB8`（青绿色）
- 用于 Header 背景、按钮、Tab 激活状态、重要标签

### 背景色
- **页面背景**：浅灰色（如 `#F5F5F5` 或 `#F0F2F5`）
- **卡片背景**：白色 `#FFFFFF`

### 文字色
- **主标题**：`#333333`
- **正文**：`#666666`
- **辅助文字**：`#999999`

## 顶部 Header

- 背景色：青绿色 `#22BDB8`
- 标题："工单处理中心"，白色文字，居中
- 高度：约 44-48px
- 状态栏适配：预留安全区域

## 底部 Tab 栏

三个 Tab 项：

| Tab | 状态 | 说明 |
|-----|------|------|
| 发起工单 | 已开通 | 点击跳转发起工单页 |
| 我的 | 已开通 | 点击跳转我的页 |
| 组内 | 未开通 | 视觉保留，点击无功能或提示"暂未开通" |

- Tab 栏背景：白色，顶部细线分隔
- 激活状态：图标和文字使用主色 `#22BDB8`
- 未激活状态：图标和文字使用灰色 `#999999`

## 发起工单页

### 入口卡片
- 白色圆角卡片，底部有阴影或边框
- 卡片内包含各工单类型入口
- 已有入口：物流工单、客服工单
- **新增入口：预算申请工单**
- 每个入口包含图标 + 名称 + 右侧箭头

## 我的页

### 搜索框
- 位于页面顶部（Header 下方或页面内）
- 圆角输入框，灰色背景
- 左侧搜索图标，placeholder "搜索工单"

### 状态 Tab
- 横向滚动 Tab 栏
- 四个 Tab：待办 / 已办 / 抄送我的 / 已发起
- 激活 Tab 底部有主色指示线

### 筛选栏
- "全部工单" 筛选按钮
- 点击展开筛选选项

### 工单卡片列表
- 纵向列表，卡片之间有间距
- 每张卡片：
  - 白底圆角
  - 左侧有状态色条或标签
  - 工单标题、编号
  - 发起时间、当前处理人
  - 状态标签（使用展示字段 `displayStatus`）

## 通用组件规范

### 卡片样式
- 背景：白色 `#FFFFFF`
- 圆角：`8px` 或 `12px`
- 内边距：`12px` 或 `16px`
- 阴影：`0 2px 8px rgba(0,0,0,0.06)` 或无边框线分隔

### 按钮样式
- 主按钮：青绿色背景，白色文字，圆角
- 次按钮：白色背景，青绿色边框，青绿色文字
- 禁用状态：灰色背景

### 表单样式
- 表单标签左对齐，输入框占满剩余宽度
- 必填项标红
- 输入框圆角，灰色边框，聚焦时主色边框

## 响应式

- 设计基准宽度：`375px`（iPhone 标准）
- 最大宽度限制：`430px`
- 居中显示，两侧留白或灰色背景延伸
