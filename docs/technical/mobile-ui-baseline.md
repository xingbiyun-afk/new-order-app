# 移动端 UI 基线 — 工单处理中心

> 本文档记录工单处理中心 App 端的移动端样式规范。
> 本系统工单存在 App 和 Web 两端，App 端移动样式将作为 Web 端非查询类工单页面的复用参考。
> 跨端复用策略详见 `cross-end-work-order-ui-strategy.md`。

## 色彩规范

### 主色
- **主色**：`#22BDB8`（青绿色）
- 用于 Header 背景、按钮、Tab 激活状态、重要标签

### 背景色
- **页面背景**：浅灰色（如 `#F5F5F5` 或 `#F0F2F5`）
- **卡片背景**：白色 `#FFFFFF`

### 文字色
- **主标题**：`#333333`
- **正文**：`#666666`
- **辅助文字**：`#999999`

## 顶部 Header

- 背景色：青绿色 `#22BDB8`
- 标题："工单处理中心"，白色文字，居中
- 高度：约 44-48px
- 状态栏适配：预留安全区域

## 底部 Tab 栏

三个 Tab 项：

| Tab | 状态 | 说明 |
|-----|------|------|
| 发起工单 | 已开通 | 点击跳转发起工单页 |
| 我的 | 已开通 | 点击跳转我的页 |
| 组内 | 未开通 | 视觉保留，点击无功能或提示"暂未开通" |

- Tab 栏背景：白色，顶部细线分隔
- 激活状态：图标和文字使用主色 `#22BDB8`
- 未激活状态：图标和文字使用灰色 `#999999`

## 发起工单页

### 入口卡片
- 白色圆角卡片，底部有阴影或边框
- 卡片内包含各工单类型入口
- 已有入口：物流工单、客服工单
- **新增入口：预算申请工单**
- 每个入口包含图标 + 名称 + 右侧箭头

## 我的页

### 搜索框
- 位于页面顶部（Header 下方或页面内）
- 圆角输入框，灰色背景
- 左侧搜索图标，placeholder "搜索工单"

### 状态 Tab
- 横向滚动 Tab 栏
- 四个 Tab：待办 / 已办 / 抄送我的 / 已发起
- 激活 Tab 底部有主色指示线

### 筛选栏
- "全部工单" 筛选按钮
- 点击展开筛选选项

### 工单卡片列表
- 纵向列表，卡片之间有间距
- 每张卡片：
  - 白底圆角
  - 左侧有状态色条或标签
  - 工单标题、编号
  - 发起时间、当前处理人
  - 状态标签（使用展示字段 `displayStatus`）

## 通用组件规范

### 卡片样式
- 背景：白色 `#FFFFFF`
- 圆角：`8px` 或 `12px`
- 内边距：`12px` 或 `16px`
- 阴影：`0 2px 8px rgba(0,0,0,0.06)` 或无边框线分隔

### 按钮样式
- 主按钮：青绿色背景，白色文字，圆角
- 次按钮：白色背景，青绿色边框，青绿色文字
- 禁用状态：灰色背景

### 表单样式
- 表单标签左对齐，输入框占满剩余宽度
- 必填项标红
- 输入框圆角，灰色边框，聚焦时主色边框

## 页面级布局

### 整体结构

```
┌─────────────────────────┐
│  Header (#22BDB8)       │  ← 48px，白色标题居中
├─────────────────────────┤
│                         │
│  Page Content           │  ← 浅灰背景，内容区
│  (卡片列表/表单等)       │
│                         │
├─────────────────────────┤
│  Bottom Tab             │  ← 56px，白底，3个Tab
└─────────────────────────┘
```

### 页面内边距
- 水平方向：`16px`（左右）
- 卡片之间间距：`12px`
- 顶部与 Header 间距：`12px`
- 底部预留 Tab 高度 + `16px`

## 工单卡片详细规范

### 卡片结构

```
┌────────────────────────────────────┐
│ [色条]  工单标题          [状态标签] │
│         编号 · 日期                 │
│         发起人 · 当前处理人          │
└────────────────────────────────────┘
```

### 卡片元素

| 元素 | 样式 |
|------|------|
| 左侧色条 | 宽 4px，高 40px，圆角 2px，颜色根据状态变化 |
| 工单标题 | 15px，#333，单行省略 |
| 工单编号 | 13px，#999 |
| 日期 | 13px，#999 |
| 状态标签 | 圆角 4px，padding 2px 8px，12px 字号 |

### 状态色对照

| 状态类型 | 色条/标签色 | 背景色 |
|----------|-------------|--------|
| 审批中 | `#22BDB8` | `#E6F8F8` |
| 待处理 | `#FF9800` | `#FFF3E0` |
| 已通过 | `#4CAF50` | `#E8F5E9` |
| 已驳回 | `#F44336` | `#FFEBEE` |
| 已完成 | `#9E9E9E` | `#F5F5F5` |

> 注：以上状态值仅作展示参考，实际使用 `displayStatus` 等展示字段，不定义正式枚举。
>
> **重要**：状态文案（"审批中""待处理""已通过""已驳回""已完成"等）仅用于 UI 样式展示，不代表正式工单状态、审批状态或后端状态码。原型阶段不涉及状态机定义，实际状态映射待后续需求明确后统一处理。

## 发起工单页入口卡片

### 结构

```
┌────────────────────────────────────┐
│  [圆形图标]  工单类型名称      >     │
│              类型描述文字            │
└────────────────────────────────────┘
```

### 样式
- 圆形图标：40x40px，背景色 `#22BDB8`，白色文字/图标
- 名称：16px，#333，font-weight 500
- 描述：13px，#999
- 右侧箭头：`>`，18px，#999
- 整张卡片可点击，active 态 opacity 0.8

### 当前入口

| 入口 | 图标文字 | 描述 | 状态 |
|------|----------|------|------|
| 物流工单 | "物流" | 发起物流相关工单 | 已有 |
| 客服工单 | "客服" | 发起客服相关工单 | 已有 |
| 预算申请工单 | "预算" | 发起预算申请工单 | **本期新增** |

## 我的页 — 完整布局

### 从上到下结构

```
┌─────────────────────────┐
│  Header (工单处理中心)   │
├─────────────────────────┤
│  [ 🔍 搜索工单 ]         │  ← 搜索框
│                         │
│  待办 | 已办 | 抄送我|已发│  ← 状态 Tab
│  ─────                  │
│  全部工单          筛选 ▼│  ← 筛选栏
│  ┌───────────────────┐  │
│  │ [绿条] 标题   [审批中]│  ← 工单卡片1
│  │ 编号 · 日期          │  │
│  └───────────────────┘  │
│  ┌───────────────────┐  │
│  │ [橙条] 标题   [待处理]│  ← 工单卡片2
│  │ 编号 · 日期          │  │
│  └───────────────────┘  │

├─────────────────────────┤
│  发起工单 | 我的 | 组内   │  ← 底部 Tab
└─────────────────────────┘
```

### 状态 Tab 交互
- 横向滚动，超出可滑
- 激活项：文字 #333，底部 2px 主色指示线，font-weight 500
- 未激活项：文字 #999，无指示线
- 切换时内容区联动刷新

### 搜索框交互
- 点击后输入框聚焦，placeholder 消失
- 支持实时搜索或点击搜索按钮
- 搜索时展示 loading 状态（骨架屏）

## "组内"入口说明

**当前未开通，不纳入本期功能范围。**

- 底部 Tab 中保留"组内"文字和图标，作为视觉占位。
- 点击后展示"暂未开通"提示页或 toast。
- 不实现任何业务功能。
- 颜色使用灰色 #999，与未激活 Tab 一致。

## 响应式

- 设计基准宽度：`375px`（iPhone 标准）
- 最大宽度限制：`430px`
- 居中显示，两侧留白或灰色背景延伸
