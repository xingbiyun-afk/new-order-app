## 2. 变更定位

本 CR 属于**发起链路治理型小切口**，用于补强当前“产品申请工单”发起链路在参考实现阶段的**场景验收样本、回填/返回边界、导入链路规则承接与文档治理**。

本 CR 的定位必须明确为：

1. **不是新增业务功能**
2. **不是新增业务规则**
3. **不是重做发起页结构**
4. **不是推翻既有预算选择、专卖店选择、产品选择、导入能力与驳回后重新发起规则**
5. **不是视觉改版**
6. **不是发起页功能增强单**
7. **不是把导入能力拆成独立专项重构**

本轮目标是在**不扩大功能范围、不抬高页面复杂度、不重开已收口概念**的前提下，使发起链路达到：

- 可系统验收
- 可稳定复现
- 可直接按样本验证
- 可直接按文档执行
- 可减少后续反复争议

## 4. 固定前提与不可变更口径

本 CR 必须严格继承以下固定背景，不得回退或改写。

### 4.1 工程与对象固定
- 主工程固定：`vue-reference/`
- 技术栈固定：Vue3
- React 归档目录：`legacy/react-reference/`，不作为当前实现依据
- 当前正式业务对象固定为：
  1. 产品申请工单
  2. 产品申请预算核销

### 4.2 禁止重新引入的旧概念
本轮实现、文档、注释、mock、页面文案中严禁重新引入以下内容：

1. 预算责任人管理
2. 主责 / 备责
3. 责任人异常影响预算核销
4. 待调整预算状态
5. 预算锁定 / 锁定状态 / 预算被锁定

说明：  
涉及驳回后重新发起时，只允许使用：

- 原预算号沿用
- 不允许切换预算号
- 不可继续基于原单重提

不得回退使用“预算锁定”类旧表达。

### 4.3 发起链路既有业务口径不得推翻
本轮不得推翻以下已收口规则：

#### 4.3.1 预算相关
1. 发起人与审批人统一来源于“产品申请审批流程管理”
2. 预算状态只有：
   - 草稿
   - 已生效
   - 已作废
   - 已到期
3. `applicationAvailable=false`：
   - 预算核销不可用
   - 前端禁选
4. `isAbnormal`：
   - 仅表示组织异常提示
   - 不阻断使用
5. 金额展示统一使用：
   - 千分符
   - 两位小数
   - “剩余额度（净额）”
   - “预算总额（净额）”
   - “本次申请额度（净额）”
6. 禁止回退为：
   - 可用金额
   - 总额
   - 预算锁定等旧口径

#### 4.3.2 工单结构相关
1. 工单结构固定为：
   - 工单头
   - 多个产品申请订单明细
   - 每个明细 = 一个订单归属专卖店
   - 多条 SKU/产品明细
2. 用户术语统一：
   - 产品申请订单明细
   - 产品
   - 添加产品
   - 添加产品申请订单
3. 同一店编预算核销产品填写在一个明细
4. 同店编多明细审批通过后生成正式订单时归并：
   - 客户层级 = 6 → 产品申请表订单
   - 客户层级 = 0-4 → 内部申请表订单

#### 4.3.3 产品 / 金额 / 数量相关
1. 产品来源于“产品申请管理菜单”
2. 菜单维护：
   - 可申请产品
   - JDE 价格
   - 是否打折
   - 折扣值
   - 可申请数量规则
3. 折扣不是常量
4. 发起页、详情页不单独展示折扣字段
5. 产品选择页可保留折扣辅助展示
6. 金额计算：
   - 打折：`JDE 价格 × 数量 × 折扣`
   - 不打折：`JDE 价格 × 数量`
7. 金额保留两位小数
8. 提交前校验预算可用金额
9. 数量控制以前端前置 + 后端兜底为准

#### 4.3.4 附件相关
1. 附件必传
2. 支持：
   - JPG
   - PNG
   - GIF
   - Word
   - PDF
   - PPT
   - Excel
3. 单文件 ≤ 20MB
4. 最多 10 个附件
5. 发起页支持：
   - 上传
   - 预览
   - 删除

#### 4.3.5 导入相关
1. 批量导入能力已存在
2. 本轮不重做底层导入能力
3. 当前导入固定字段顺序为：
   1. 产品编号
   2. 数量
   3. 专卖店编号
4. 模板固定为：
   - 无表头
   - 第 1 行即数据
5. 文件导入支持：
   - `.xlsx`
   - `.xls`
   - `.csv`
6. 当前导入校验已区分：
   - 硬错误（error）
   - 软警告（warning）
7. 本轮只做：
   - 场景补强
   - 规则承接补强
   - 错误示例补强
   - 验收样本补强
8. 本轮不做：
   - 新增导入格式
   - 替换 Excel 解析方案
   - 导入能力专项重构
