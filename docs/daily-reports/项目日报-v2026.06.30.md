## 第一部分：Vue3 参考实现迁移（CR-20260630-001）

### 工程初始化
- 在 `vue-reference/` 目录下初始化 Vue3 + Vite + TypeScript 工程
- 安装 Vue Router 4、Pinia、Tailwind CSS 4
- 配置 `vite.config.ts`（Vue 插件 + Tailwind 插件 + `base: './'`）

### 页面迁移（8 个页面）
- **发起工单入口页**（`/create`）— 物流/客服/产品申请三个入口 + Tab
- **我的列表页**（`/my`）— 搜索框 + Tab + 筛选 + 键值对卡片
- **组内页**（`/team`）— 未开通占位
- **产品申请发起页**（`/product-apply/create`）— 预算/申请信息/分组/产品/附件/提交
- **预算选择页**（`/product-apply/budget-select`）— 搜索过滤 + 预算卡片 + 异常提示
- **专卖店搜索页**（`/product-apply/store-select`）— 搜索 + 列表 + 回填
- **产品搜索页**（`/product-apply/product-select`）— 搜索 + 卡片 + 回填
- **详情页**（`/product-apply/detail/:id`）— 基础信息/审批流/订单结果/审批操作

### P1 特性承接
- 分组折叠/展开、跨分组 SKU 校验、动态数量提示
- 关联订单独立卡片、失败原因分层展示
- URL Search Params 跨页数据传递、sessionStorage 持久化

### Bug 修复
- **修复1**：CDN 部署 chunk 加载问题（`import()` → 同步 `import`）
- **修复2**：专卖店/产品选择回填失败（`computed`+`watch` → 直接 `watch(() => route.query)` + `immediate: true`）
