## 本轮新增规则点

### 1. 审批超时自动驳回规则
- 适用对象：产品申请工单审批中的任一审批环节
- 超时阈值：14 天
- 起算点：进入当前待审批节点时刻
- 处理结果：系统自动驳回，工单主状态为"已驳回"
- 页面承接：详情页驳回原因需体现"审批超时自动驳回"语义

### 2. 组织变更场景在途工单处理规则
- 审批过程中相关组织信息发生变更时，由财务通知相关环节审批人驳回
- 财务通知本身不改变工单状态
- 通知后 14 天内审批人执行驳回 → 按人工驳回落定
- 通知后 14 天仍无人处理 → 按系统自动驳回兜底
- 不新增主状态，最终仍为"已驳回"

### 3. 财务通知驳回与系统自动驳回关系
- 财务通知是流程推动动作，不是审批结果
- 审批人执行驳回 = 人工驳回
- 超时无操作 = 系统自动驳回
- 两者不可混淆展示

### 4. 驳回原因来源分类（4 类）
| 分类 | 执行者 |
|------|--------|
| 普通审批驳回 | 审批人 |
| 审批超时自动驳回 | 系统 |
| 组织变更后人工驳回 | 审批人（受财务通知） |
| 组织变更后超时自动驳回 | 系统 |
