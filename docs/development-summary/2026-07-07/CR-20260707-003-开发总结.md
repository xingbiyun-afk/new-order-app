- 在 `getMockWorkOrderDetail()` 函数前新增「详情页场景映射总表」注释块，集中维护 scene key ↔ id ↔ 工单号 ↔ 主状态的映射关系

  - 场景10（completed-store-change）：验证专卖店变更 + 实际专卖店展示 + 备注说明
