## 一、变更背景

截至 2026-06-30，产品申请工单 Vue 参考实现已完成首轮迁移（CR-001）、一致性修正（CR-002）、"我的"页列表补强（CR-003）和发起页专卖店交互收尾（CR-004）。

当前阶段主工作重心转向产品申请工单发起页，围绕新增需求、交互优化和页面细节继续收敛。本次变更涵盖：申请信息区压缩、命名口径统一、说明文案添加、附件区升级、PDF 合并提示，同步更新 4 份产品文档。

### 3.1 申请信息区压缩

**文件**：`vue-reference/src/views/work-order-budget/CreateView.vue`

**修改内容**：
- 预算未选中时仅显示"申请人"一行，卡片标题右侧显示"选择预算后展开"

**验证结果**：

2. 选择预算后 → 展开完整三行 ✅

**影响范围**：仅模板层条件渲染，不影响数据逻辑。
