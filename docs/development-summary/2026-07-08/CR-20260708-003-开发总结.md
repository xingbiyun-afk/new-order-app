## 开发内容

### 1. 草稿链路摘要层（DetailView.vue）

**新增辅助函数**：
- `getDraftLinkResultLabel(link)`：获取链路结果标签（订单已创建/草稿提交失败/审核驳回/草稿已删除）
- `getDraftLinkLatestTime(link)`：获取链路最近一次时间
- `isCurrentDraftLink(link)`：判断是否为当前承接结果的链路
- `sortDraftLinks(links)`：排序草稿链路（当前链路优先，历史链路后置）

**模板修改**：
- 历史入口展开后，草稿链路默认只展示摘要层（draftId + 结果标签 + 最近时间 + 展开箭头）
- 链路详情需用户再次点击草稿链路头才展开
- 多条草稿链路时使用 `sortDraftLinks` 排序，当前链路优先

### 2. 兼容旧结构（retryHistory）
- 旧结构同样遵循摘要层 + 详情层分离
- 旧结构链路默认标记为 `draft-link-old` 弱化样式

### 3. 视觉减负（CSS）

| 元素 | 修改前 | 修改后 |
|------|--------|--------|
| 历史区失败项背景 | #FFEBEE（强红底） | #f8f8f8（浅灰底）+ 2px 左边细红边框 |
| 历史区删除原因 | 强红块 | 中性说明：灰色斜体 + transparent 背景 |
| 旧链路 | 与当前链路等权 | 更轻边框 + 更淡背景 + 更低文案层级 |
| 链路结果标签 | 统一强度 | 历史链路 opacity: 0.7 弱化 |
