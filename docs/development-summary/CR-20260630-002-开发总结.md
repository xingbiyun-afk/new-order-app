```

├── types/                    # 类型定义
│   └── index.ts              # Budget/ProductItem/StoreGroup/ApprovalNode 等
```

### 4.2 状态管理设计

| 状态 | 位置 | 说明 |
|------|------|------|
| 预算选择 | Pinia Store | `selectedBudget` |
| 专卖店分组 | Pinia Store + sessionStorage | `storeGroups` 持久化 |
| 折叠状态 | Pinia Store | `collapsedGroups` |
| 附件列表 | 组件本地状态 | `attachments` |
| Toast | 组件本地状态 | `toast` |

### 4.3 跨页面数据传递

延续 React 参考实现的 URL Search Params 方案：
- 发起页 → 选择页：通过路由跳转
- 选择页 → 发起页：通过 URL query 参数回填
- 发起页读取 `budgetId`、`storeCode`、`storeName`、`groupId`、`productCode` 等参数

## 五、React → Vue 迁移映射

| React 概念 | Vue 等价物 |
|-----------|-----------|
| useState | ref / reactive |
| useMemo | computed |
| useEffect + useCallback | watch / onMounted |
| useNavigate | useRouter |
| useSearchParams | useRoute().query |
| useLocation | useRoute |
| React Context | Pinia Store |
| JSX {condition && } | v-if |
| JSX {list.map()} | v-for |
| onClick={() => fn()} | @click="fn()" |
| style={{ color: '#333' }} | :style="{ color: '#333' }" |
