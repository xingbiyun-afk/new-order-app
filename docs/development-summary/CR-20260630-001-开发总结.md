## 六、Bug 修复记录

### 修复1：CDN 部署 chunk 加载问题

**问题**：懒加载路由 chunk 在 CDN 部署环境下路径解析失败，页面空白。  
**根因**：`import()` 动态导入生成的 chunk 文件在 CDN 静态部署时路径解析错误。  
**修复**：将路由组件从 `import()` 动态导入改为同步 `import`，打包为单一 bundle。  
**提交**：`47875bc`

### 修复2：专卖店/产品选择回填失败

**问题**：从专卖店选择页或产品选择页返回发起页时，已选择的内容未回填到表单。  
**根因**：Vue Router 的 `route.query` 是普通对象，通过 `computed` 间接创建 ref 再 `watch` 监听，无法正确触发响应式更新。  
**修复**：改为直接 `watch(() => route.query, ...)` 并添加 `immediate: true`，确保组件挂载时立即检查 URL 参数。回填完成后 `router.replace({ query: {} })` 清除参数。  
**提交**：`e424445`
