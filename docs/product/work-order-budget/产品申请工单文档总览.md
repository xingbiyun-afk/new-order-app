# 产品申请工单文档总览

## 1. 文档说明

本文档是产品目录 `docs/product/work-order-budget/` 的总览入口，用于说明当前主文档、页面规则文档、辅助文档与协作文档的层级关系，指导阅读顺序。

本文档不直接承载业务细则。

## 2. 当前正式业务对象

- **产品申请工单**
- **产品申请预算核销**

历史"预算申请工单"等表述已废弃，不再作为现行正式命名。
