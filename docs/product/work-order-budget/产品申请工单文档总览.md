# 产品申请工单文档总览（v1.1）

## 1. 文档说明

本文档是产品目录 `docs/product/work-order-budget/` 的总览入口与阅读导航文档，用于说明当前主文档、页面规则文档、辅助文档与协作文档的层级关系和推荐阅读顺序。

本文档不直接承载业务细则。

## 2. 当前正式业务对象

- **产品申请工单**
- **产品申请预算核销**

历史"预算申请工单"等表述已废弃，不再作为现行正式命名。

## 9. 不同角色阅读建议

| 角色 | 阅读路径 | 说明 |
|---|---|---|
| 产品 / 需求 | 总览 → PRD → 流程与状态规则 → 页面规则（按需） | 以业务全貌和规则主线为先，页面级细节按需查阅 |
| 前端开发 | 总览 → AI 开发任务说明 → 页面结构草图 → 对应页面规则明细 | 以开发约束和页面结构为先，细则以页面规则文档为准 |
| 测试验收 | 总览 → 页面规则明细（对应页面）→ 流程与状态规则 | 以页面级交互和状态覆盖为先，横切规则用于验证边界 |
| AI 协作 | 总览 → AI 开发任务说明 → 对应页面规则明细 | 以开发边界和页面直接执行依据为先 |

## 10. 文档角色分工说明

| 文档类别 | 角色定位 | 代表文档 |
|---|---|---|
| 主文档 | 业务总文档，定义背景、目标、范围、主链路与总规则 | PRD、流程与状态规则、预算核销规则、审批承接与异常处理说明 |
| 页面规则文档 | 页面执行细则，为对应页面开发与验收的直接依据 | 创建页/预算选择页/详情页/我的页面规则明细 |
| 辅助文档 | 结构/字段视角的补充，不替代主文档和页面规则文档 | 字段清单、页面结构草图说明 |
| 协作文档 | AI 与开发治理约束，定义任务范围、协作边界与开发约束 | AI 开发任务说明 |

### 10.1 重要提醒

- **页面规则文档仍是页面直接执行依据**，PRD 不替代页面规则文档；
- **辅助文档不替代主文档和页面规则文档**，仅作为结构/字段视角的补充参考；
- **协作文档不替代业务规则文档**，仅定义 AI 开发边界和协作约束。
