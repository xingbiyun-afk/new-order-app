# 预算申请工单 — 产品说明

## 背景

在现有"工单处理中心"基础上新增"预算申请工单"入口，为用户提供预算申请的发起、查看和审批功能。

## 已有功能

### 发起工单页
- 物流工单（已有）
- 客服工单（已有）
- **预算申请工单（本期新增入口）**

### 我的页
- 搜索框
- Tab：待办 / 已办 / 抄送我的 / 已发起
- 全部工单筛选
- 工单卡片列表

### 底部 Tab
- 发起工单
- 我的
- 组内（未开通，仅视觉占位）

## 本期范围

### 包含
- 预算申请工单入口（发起工单页新增入口卡片）
- 预算申请表单页（目录占位）
- 预算申请详情页（目录占位）
- 审批流程相关页面（目录占位）

### 不包含
- 组内功能
- 复杂 Mock 数据
- API 对接
- 正式状态枚举定义

## 展示字段

原型中使用以下展示字段（不定义正式枚举）：
- `displayStatus` — 展示状态文本
- `approvalStatusText` — 审批状态文本
- `flowStatusText` — 流程状态文本

## 样式要求

预算申请工单除表单内容不同外，展示风格和流程沿用现有工单样式。
参考 `docs/technical/mobile-ui-baseline.md` 了解完整 UI 规范。
