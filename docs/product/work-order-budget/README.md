# 产品申请工单 / 产品申请预算核销 — 产品说明

> 历史说明：本目录早期命名为 `work-order-budget`，用于承接"预算申请工单"原型规划。
> 根据当前最新业务口径，后续统一按"产品申请工单 / 产品申请预算核销"理解。

## 背景

在现有"工单处理中心"基础上，新增"产品申请工单"入口。产品申请工单主要用于预算核销。

## 已实现功能（Vue 参考实现）

### 发起工单页
- 物流工单（已有）
- 客服工单（已有）
- **产品申请工单**（P0+P1 完整实现）
  - 申请信息区：预算未选时压缩，选定后展开
  - 产品申请订单明细：按专卖店拆分，支持添加/折叠/展开
  - 产品选择：搜索 + 卡片列表 + 数量校验
  - 附件上传区：网格布局 + PDF 合并提示
  - **批量导入**：文本粘贴 / Excel/CSV 文件 / 模板下载（CR-20260701-002）
  - 提交校验：金额不超预算可用金额
  - **发起页体验优化**（CR-20260702-002）：无草稿规则、预算/申请信息区压缩展开、滚动预算摘要条、收起态小计、产品术语统一、附件必传感知、提交禁用态、金额预算关系提示、数量提示优化、操作层级区分

### 预算选择页
- **卡片信息层级优化**（CR-20260702-001）：预算标签机制、排序规则、搜索过滤、选中态/不可选态/空态
- 异常状态提示

### 专卖店搜索选择页
- 搜索 + 列表选择 + 回填
- **按 App 现有页面 1:1 回归**（CR-20260703-002）：搜索栏（门店信息标签+搜索框+取消按钮）+ 搜索历史记录 + 行式列表

### 产品搜索选择页
- 搜索 + 卡片列表 + 回填（P1-01）

### 详情页
- 状态驱动首屏差异化（CR-20260706-001）：处理中新增当前审批进度卡片；已驳回突出驳回信息区；已结束订单结果区前移
- 区块顺序：状态→驳回/审批进度→基础→发起→预算→明细→附件→审批流/订单结果→操作
- 产品明细卡片头：两行布局（标题+编号 / 款数·件数+分组金额）
- 汇总行：四列网格（笔订单/专卖店/款/件）
- 统一提示风格（.tip-banner + error/warning/info 三种变体）
- 审批流节点字段化与视觉区分（CR-20260706-002）：`nodeName` 优先展示，节点类型色彩区分（当前橙色/通过/驳回红色/发起青绿色）
- 预占订单 chip 胶囊平铺展示，最终审批通过节点不展示
- 失败原因去重（`failReasons: string[]` + `dedupFailReasons()`）
- 折叠重试历史（`OrderAttempt` + `retryHistory`）
- 专卖店变更：实际专卖店与工单明细不一致时展示「实际专卖店（订单）」橙色警告 + 订单备注灰色斜体（CR-20260706-004）
- 失败原因类型明确：产品库存不足 / 专卖店下单标识关闭 / 订单审核驳回草稿（3 类）
- 移除导航栏右上角状态字（CR-20260706-003）
- `scrollBehavior` 修复：列表页→详情页不再继承滚动位置（CR-20260706-003）
- Mock 场景覆盖 11 个 id（1-9 基础 + 10 全部失败 + 11 专卖店变更）
- **预占订单展示边界（CR-20260706-003 验收后最终口径）**：预占订单仅在审批流**发起节点**展示，订单结果区不承接预占订单
- **订单结果状态固定口径（CR-20260707-002）**：已创建 / 客服已审核 / 财务已审核 / 草稿，不再使用"已生成/生成成功"
- **重试历史展示边界（CR-20260707-002）**：默认折叠，外层只承接最新结果摘要，历史失败原因仅在折叠区展示；"生成尝试历史"文案统一为"订单创建历史"
- **布局修复（CR-20260707-002）**：flex column 100vh 布局确保 `.page-content` 为唯一滚动容器，修复双重滚动条 + 页面切换左移问题
- **场景验收样本总表**（CR-20260707-003）：10个正式场景（scene key / id / 工单号 / 主状态 / 核心验证点 / 不应出现的错误 / 推荐访问方式），支撑按表验收、截图、走查、AI 自检
- **Mock 入口访问说明**（CR-20260707-003）：支持 `/detail/:id` 与 `?scene=xxx` 双入口，query scene 优先级高于 id 路由
- **页面容器与滚动承接规则**（CR-20260707-003）：`.page-content` 为唯一主滚动容器，App.vue 显式处理路由切换滚动复位
- **详情页现行规则直接执行依据**：《产品申请工单详情页面规则明细.md》（**v0.4 · CR-20260707-003**）
- **历史描述与现行规则优先级**：如与历史阶段性描述不一致，以 CR-20260706-003 验收后口径及规则明细文档现行规则为准
