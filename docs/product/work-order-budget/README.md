## 9. 已实现功能（Vue 参考实现）

### 发起工单页
- 申请信息区压缩/展开、产品申请订单明细按专卖店拆分
- 产品选择（搜索 + 卡片列表 + 数量校验）、附件上传（网格布局 + PDF 合并提示）
- 批量导入（文本粘贴 / Excel/CSV 文件 / 模板下载）
- 提交校验、预算/信息区交互、产品术语统一、提交禁用态

### 预算选择页
- 卡片信息层级（标签机制/排序规则/搜索过滤/选中态/不可选态/空态）

### 专卖店搜索选择页
- 搜索 + 历史记录 + 列表选择 + 回填

### 产品搜索选择页
- 搜索 + 卡片列表 + 回填

### 详情页
- 状态驱动首屏差异化（处理中/已驳回/已结束）
- 区块顺序：状态→驳回/审批进度→基础→发起→预算→明细→附件→审批流/订单结果→操作
- 产品明细卡片头两行布局、汇总行四列网格、统一提示风格
- 审批流节点字段化与视觉区分、预占库存订单 chip 展示
- 订单结果区展示模型（专卖店结果单元 / 5类枚举 / draftId 聚合 / 信息变更）
- 重试历史折叠展示、专卖店变更识别、失败原因去重
- Mock 覆盖 12 个场景
- 详情页规则 v0.9、创建页规则 v0.3

### 我的列表页
- Tab（待办/已办/已发起）、卡片标题+三行正文+时间

> 现行页面执行细则以对应页面规则明细文档为准。页面上方各项能力的历史版本演进详见下方 CR 追溯表。

- `displayStatus` — 展示状态文本
- `approvalStatusText` — 审批状态文本
