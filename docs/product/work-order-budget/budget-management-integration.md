# 产品申请工单与预算管理集成说明

## 1. 预算管理来源

预算管理规则来源于 new-order-system 的预算管理模块。

App / Vue 端不重做预算管理。

## 2. 预算状态

当前有效预算状态：

- 草稿
- 已生效
- 已作废
- 已到期

不存在“待调整”。

## 3. 预算核销可用性

`applicationAvailable` 是预算核销可用标识。

产品申请工单可用预算号至少应满足：

- 预算状态为已生效
- applicationAvailable = 可申请
- 可用金额 > 0
- 发起人具备预算核销资格

## 4. 预算核销资格

预算核销资格由以下条件共同决定：

1. 预算归属组织：员工所属组织必须在预算归属组织范围内；
2. 预算适用范围：员工角色必须在预算适用范围内。

不存在独立预算核销资格名单。

## 5. isAbnormal 语义

`isAbnormal` 仅表示组织异常相关影响标识。

不表示：

- 审批人员异常
- 预算不可核销
- 正式业务规则

## 6. 发起人与审批人

发起人与审批人规则来源于产品申请审批流程管理。

不再来源于预算责任人管理。

预算责任人体系已下线。
