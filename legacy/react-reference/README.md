# legacy/react-reference — 历史 React 参考工程

> ⚠️ **本目录为历史 React 参考工程，非当前主工程。**

---

## 目录定位

| 项 | 说明 |
|----|------|
| 定位 | 历史阶段 React 参考工程 |
| 状态 | **归档 / 非当前主工程** |
| 用途 | 移动端样式对照、交互回溯、历史参考 |
| 当前主方向 | Vue / Vue3 |
| 当前主目录 | `vue-reference/` |

---

## 重要提示

1. **不得基于本目录判断当前仓库主技术栈。**
2. 当前正式承接方向已切换为 Vue / Vue3，当前主实现目录为 `vue-reference/`。
3. 本目录仅用于历史参考，不作为后续正式开发承接方向。
4. 如需运行历史 React 工程，使用根目录快捷脚本：
   ```bash
   npm run dev:react-legacy
   npm run build:react-legacy
   ```

---

## 运行命令

```bash
# 进入本目录
cd legacy/react-reference

# 安装依赖（首次）
npm install

# 开发
npm run dev

# 构建
npm run build
```
