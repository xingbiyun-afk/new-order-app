# app/layout — 布局组件

## 说明

本目录包含应用的布局组件，如底部 Tab 栏、页面容器等。

## 布局结构

```
App
├── Header（顶部标题栏）
├── Page Content（页面内容）
└── Bottom Tab（底部 Tab 栏）
```

## 底部 Tab

| Tab | 路径 | 状态 |
|-----|------|------|
| 发起工单 | `/create` | 已开通 |
| 我的 | `/my` | 已开通 |

## 注意事项

- 移动端优先设计。
- Header 使用主色 `#22BDB8` 背景。
- 底部 Tab 未激活项使用灰色。
