# work-order-center — 工单处理中心

## 说明

工单处理中心是已有功能模块，包含发起工单页和我的页。

## 已有功能

### 发起工单页
- 入口卡片列表
- 已有入口：物流工单、客服工单
- 预留入口：预算申请工单（由 `work-order-budget` 模块实现）

### 我的页
- 搜索框
- 状态 Tab（待办/已办/抄送我的/已发起）
- 全部工单筛选
- 工单卡片列表

## 目录结构

```
work-order-center/
├── README.md
├── pages/         # 页面组件
├── components/    # 模块内组件
├── services/      # API 服务（预留）
├── hooks/         # 自定义 Hooks
├── mocks/         # Mock 数据（预留）
├── types/         # 类型定义
└── utils/         # 工具函数
```

## 注意事项

- 本阶段不开发具体业务页面。
- 样式沿用现有工单处理中心风格。
- 状态使用展示字段，不定义正式枚举。
