# new-order-app Vue 参考实现

本目录为 `new-order-app` 项目的 **Vue3 + TypeScript + Vite 参考实现**，初始迁移自 React 参考实现（`src/features/work-order-budget/`，CR-20260629-001），后续经多轮迭代优化（截至 CR-20260709-001）。

## 定位

- 供正式 App Vue 工程参考样式、交互、页面结构和状态管理方案
- 不作为正式 App 生产代码
- 独立工程，与父目录 React 代码无依赖关系
- **最新迭代**：CR-20260709-001（发起链路场景验收样本补强、回填边界收口与文档治理完善：创建页规则明细 v0.1→v0.2，53 场景覆盖 A-H 八大类；CreateView.vue 驳回重提 session 隔离 + 附件上限硬拦截）

## 技术栈

- Vue 3.5 + TypeScript
- **构建工具**：Vite 8.1 + @vitejs/plugin-vue（`@tailwindcss/vite` 插件已移除）
- Vue Router 4（Hash 模式）
- Pinia（状态管理）
- 基础 CSS（`@tailwindcss/vite` 已在 CR-20260703-003 中移除，项目组件未使用 Tailwind utility class）

## 页面清单

| 路由 | 页面 | 说明 |
|------|------|------|
| `/create` | 发起工单入口页 | 物流/客服/产品申请三个入口 + 底部 Tab |
| `/my` | 我的列表页 | 搜索+Tab+筛选+键值对卡片 |
| `/team` | 组内页 | 未开通占位 |
| `/product-apply/create` | 产品申请发起页 | 预算/申请信息/产品申请订单明细/附件/提交 |
| `/product-apply/budget-select` | 预算选择页 | 搜索过滤+预算卡片 |
| `/product-apply/store-select` | 专卖店搜索页 | 搜索+列表+回填 |
| `/product-apply/product-select` | 产品搜索页 | 搜索+卡片+回填 |
| `/product-apply/detail/:id` | 详情页 | 状态驱动首屏差异化（处理中/已驳回/已结束）；审批流节点字段化+视觉区分；预占库存订单仅在发起节点 chip 展示，订单结果区不承接预占库存订单；订单结果状态统一为"已创建/客服已审核/财务已审核/草稿"；**订单结果区按专卖店结果单元展示**，含当前结果说明 5 类枚举（草稿提交失败/审核驳回/草稿已删除/订单已创建/订单已创建（信息变更））；**卡片级统一"订单创建历史"入口**，按 `draftId` 聚合草稿链路；专卖店变更/产品变更/双重变更场景（信息变更）；订单生成历史按订单号/draft 聚合，默认折叠（§6.10.13）；草稿删除原因走 outerRemark 通道；失败原因去重；**场景验收样本总表**覆盖12个正式场景；Mock 双入口（`/detail/:id` + `?scene=xxx`）；详情页规则以《产品申请工单详情页面规则明细.md》（**v0.7 · CR-20260708-003**）为直接执行依据 |

## 开发命令

```bash
npm install      # 安装依赖
npm run dev      # 开发服务器
npm run build    # 生产构建
npm run preview  # 预览构建
```
